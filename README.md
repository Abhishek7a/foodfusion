Food Fusion - A Full-Stack Food Ordering Website

Food Fusion is a full-stack web application for food ordering, built with React for the frontend, MongoDB Atlas for the database, and hosted on AWS cloud platform. It provides users with a seamless shopping experience, including features like signup, login, profile dashboard, and a shopping cart.

Features
User signup and login authentication
Profile dashboard for users
Shopping cart functionality
Browse and order food items
MongoDB Atlas for database storage
Hosted on AWS cloud platform

Technologies Used
Frontend:

React
HTML/CSS
JavaScript

Backend:

Node.js
Express.js

Database:

MongoDB Atlas
Cloud Platform:

AWS (Amazon Web Services)
